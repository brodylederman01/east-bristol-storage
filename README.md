# East Bristol Storage Website

## Preview locally
1. Open this folder in Visual Studio Code.
2. Install the **Live Server** extension by Ritwick Dey.
3. Right-click `index.html` and choose **Open with Live Server**.
4. Edit `index.html` or `styles.css`, save, and the browser will refresh.

## Files
- `index.html` — content, pricing, links, phone, email
- `styles.css` — desktop and mobile layout
- `script.js` — mobile navigation
- `logo.png` — company logo
- `facility-exterior.png` — hero image
- `unit-interior-wide.png` and `unit-interior.png` — property gallery

## Mobile support
The layout automatically changes below 760px wide:
- hamburger navigation
- stacked feature points
- one unit card per row
- full-width call buttons
- stacked contact and footer sections
