UPLOAD TO GITHUB

1. Open your East Bristol Storage repository.
2. Upload all files in this folder to the repository root.
3. When GitHub asks about duplicate files, replace the existing versions.
4. Commit the changes directly to the main branch.
5. Wait 1-5 minutes for GitHub Pages to redeploy.

The files index.html, styles.css, and script.js must remain at the top level.
The PNG image files must also remain at the top level because the website references them by filename.
Do not upload the ZIP itself as the website content.
